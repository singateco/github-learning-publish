# Cookie

https://en.wikipedia.org/wiki/HTTP_cookie

## jsp 에서 쿠키 사용법

1. 쿠키 객체 생성
2. 쿠키에 속성 값 저장.
3. addCookie()를 호출해 쿠키 추가

### 쿠키 생성

```jsp
<%
	// 쿠키 객체 생성
	Cookie c = new Cookie("id", "soldesk");

	// 유효 시간 설정
	c.setMaxAge(60*60);
	
	// 클라이언트에 쿠키 전송
	response.addCookie(c);
%>
```
