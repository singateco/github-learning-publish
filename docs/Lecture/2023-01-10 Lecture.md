# JSP Directives

- page : 해당 jsp페이지 전반 환경 설정
- include : 현재 페이지에 다른 파일의 내용을 삽입할 때 사용.
- taglib : 태그 라이브러리에서 태그를 꺼내와 사용할 수 있는 기능 제공.
